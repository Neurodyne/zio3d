# ZIO 3D

An experimental 3D game built with [ZIO](https://zio.dev/) and OpenGL ([LWJGL](https://www.lwjgl.org/)).

## Quick start

In build.sbt, ensure `lazy val os =` is set to your operating system.

`sbt run`
